# erin.recachinas.dev

Personal website for Erin Recachinas, built with [Next.js](https://nextjs.org) and deployed to [GitHub Pages](https://pages.github.com).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `master` automatically build and deploy via GitHub Actions.
